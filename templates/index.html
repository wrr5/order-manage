登录接口
POST http://192.168.0.40:8080/api/login

请求示例
{ "phone_mumber": "18031938773", "real_name": "杨依", "user_type": "2", "period_zbid": "1536636585" "password": "123456" }

创建用户接口
POST http://127.0.0.1:8080/api/users

请求示例
{ "phone_mumber": "13288886666", "real_name": "张三丰", "user_type": "1", "period_zbid": "1536636585", "password": "123456", "vz_store_id": "", "vz_factory_id": "" }
user_type(1客服, 2门店员工, 3工厂员工), 如果是门店员工则会调用微赞接口验证信息有效性。vz_store_id与vz_factory_id为外键, 可选参数, 其他必填。

查询用户接口
GET http://127.0.0.1:8080/api/users

请求示例
查询所有用户 http://127.0.0.1:8080/api/users
查询指定类型用户 http://127.0.0.1:8080/api/users?user_type=1
需要携带authorization请求头, user_type(1客服, 2门店员工, 3工厂员工), 可选参数, 查询指定类型的所有用户, 不传则查询所有类型的用户。
